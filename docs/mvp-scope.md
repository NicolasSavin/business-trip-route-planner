# Границы MVP

## Входит

- Поиск маршрутов по России на mock-данных.
- Прямые маршруты и маршруты максимум с одной пересадкой.
- Поездные и автобусные сегменты.
- Проверка доступности мест для всей группы.
- Русскоязычный интерфейс поиска.
- Демо-режим frontend без запущенного backend.
- Автоматические тесты маршрутного сервиса.

## Не входит

- Реальные API РЖД, Яндекс Расписаний или автобусных сервисов.
- Оплата, покупка или бронирование билетов.
- Авторизация и роли пользователей.
- PostgreSQL, Redis, Docker, Telegram и другие инфраструктурные сервисы.
- Хранение персональных данных сотрудников.
- Расчёт стоимости поездки.

## Mock-маршруты

- Москва → Екатеринбург, прямой поезд.
- Москва → Казань → Екатеринбург, поезд + автобус.
- Москва → Самара → Екатеринбург, автобус + поезд.
- Москва → Нижний Новгород → Екатеринбург, маршрут с недостатком мест на автобусном сегменте.
